为什么选择 mk-motion?

不仅仅是动画库，而是完整的动画驱动 UI 系统

⚡

弹簧物理引擎

基于真实物理的弹簧动画，比贝塞尔曲线更自然流畅。支持 stiffness、damping、mass 参数调节。

👆

手势交互

开箱即用的拖拽、滑动、捏合、磁吸等手势。支持移动端触控和桌面端鼠标事件。

📜

滚动触发动画

元素进入视口时自动触发动画。支持 Parallax 视差效果和滚动进度驱动动画。

🎨

设计令牌系统

通过 CSS 变量实现一致的颜色、间距、圆角。支持暗色模式和主题切换。

♿

无障碍支持

自动检测 prefers-reduced-motion，尊重用户动画偏好。完整的焦点管理支持。

🧩

Vue 3 深度集成

30+ 企业级 Vue 3 组件，支持组合式 API。Nuxt 3 模块和 Vite 插件零配置使用。

简洁优雅的 API

几行代码即可实现复杂的动画效果

import { springTo, flip, fadeIn } from '@luanlu/mk-motion/motion'
import '@luanlu/mk-motion/css'

// 弹簧物理动画
springTo(element, {
  to: { x: 0, opacity: 1 },
  stiffness: 180,
  damping: 22,
})

// FLIP 布局动画
await flip(listElement, () => {
  items.reverse()
  render()
})

// CSS 预设动画
await fadeIn('.box', { duration: 400 })
